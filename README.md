# Todo App

A simple, elegant TODO application built with vanilla HTML, CSS, and JavaScript.

## Features
## Features
- **Subtasks**: Break down tasks into smaller steps.
- **Search & Filter**: Quickly find tasks by text or status.
- **Gamification**: Enjoy satisfying completion sounds and confetti celebrations.
- **Progress Tracking**: Visualize your daily progress with a dynamic bar.
- **Personalization**: Switch between Dark and Light themes.
- **Add Tasks & Details**: Add tasks with due dates and categories.
- **Edit Tasks**: Double-click text or click the pen icon to edit tasks.
- **Drag & Drop**: Reorder tasks by dragging them.
- **Mark as Complete**: Toggle tasks as completed or active.
- **Delete Tasks**: Remove tasks you no longer need.
- **Persistence**: Tasks and theme preference are saved automatically.
- **Premium Design**: Modern aesthetics with glassmorphism and smooth animations.

## How to Run
Simply open `index.html` in any modern web browser.
