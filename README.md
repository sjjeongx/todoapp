# Todo App

A simple, elegant TODO application built with vanilla HTML, CSS, and JavaScript.

## Features
- **Add Tasks**: Quickly add new tasks to your list.
- **Mark as Complete**: Toggle tasks as completed or active.
- **Delete Tasks**: Remove tasks you no longer need.
- **Persistence**: Tasks are saved in your browser's local storage, so you don't lose them on refresh.
- **Premium Design**: Features a modern dark mode with glassmorphism effects and smooth animations.

## How to Run
Simply open `index.html` in any modern web browser.
